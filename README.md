# ☁️ Azure Pricing & FinOps Guide: Análise Estratégica de Custos

Este repositório contém uma análise aprofundada sobre gerenciamento de custos e estratégias de **FinOps** no ambiente **Microsoft Azure**, desenvolvido para o Bootcamp da **DIO**. O projeto utiliza a inteligência artificial **NotebookLM** para sintetizar documentações complexas da Microsoft em insights visuais e práticos.

---

## 📑 Documentação Principal (PDF)

O coração deste projeto é o guia consolidado abaixo. Clique no botão para visualizar ou baixar o material detalhado:

[![Download PDF](https://img.shields.io/badge/Consultar_PDF-Azure_FinOps_Mastery-blue?style=for-the-badge&logo=adobe-acrobat-reader&logoColor=white)](./Azure_FinOps_Mastery.pdf)

### Resumo do que você encontrará no guia:
- Estratégias comparativas de preços (Pay-as-you-go, Reservado, Spot).
- Modelos de Governança e tags de custo.
- Análise de TCO (Total Cost of Ownership) para migrações.
