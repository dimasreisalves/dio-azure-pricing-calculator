# ☁️ Azure Pricing & FinOps Guide

Este repositório contém um estudo aprofundado sobre a calculadora de custos do **Microsoft Azure** e estratégias de **FinOps**, desenvolvido para o Bootcamp da **DIO**. O projeto utiliza inteligência artificial (NotebookLM) para sintetizar documentações complexas em insights práticos de economia.

---

## 📑 Conteúdo do Projeto

O projeto é composto por três pilares principais de análise:

### 1. Guia de Maestria FinOps (`Azure_FinOps_Mastery.pdf`)
Um documento consolidado detalhando:
- **Estratégias de Redução de Custos:** Diferenças entre instâncias reservadas e spot.
- **Modelos de Governança:** Como aplicar Tags e políticas de custo.
- **TCO (Total Cost of Ownership):** Análise de migração on-premises para nuvem.

## 🛠️ Ferramentas Utilizadas

## 🧠 Metodologia de Estudo
Para este projeto, a documentação oficial da Microsoft foi fornecida ao NotebookLM como fonte de dados primária. A IA auxiliou na criação de resumos executivos e na organização dos tópicos de maior impacto financeiro para empresas que utilizam a nuvem Azure.
